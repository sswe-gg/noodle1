介紹拉麵起源

最初是起源於中國的麵料理，之後在日本獨自發展而成，拉麵的麵條不僅粗細・捲曲程度・加水率各為不同，連湯頭及食材也有各式各樣種類。拉麵的起源有很多種說法，起源之一是在1859年，為了與他國進行貿易往來而開港，因而有許多國外的飲食文化流傳進日本。此也是中國麵類料理傳進日本的契機。

熱門拉麵產品


醬油拉麵

將以醬油調味的醬汁作為湯頭使用。有採用雞骨及蔬菜來萃取高湯作為基底的清爽口味，也有採用豚骨為基底的濃厚口味。

Last updated 3 mins ago
味噌拉麵

將以味噌調味的醬汁作為湯頭使用。味噌拉麵的最大特色為濃厚且深層的濃密口感。據說是1961年時，由北海道札幌市的人氣拉麵店「味之三平」的店主最先研發而成的。

Last updated 3 mins ago
豚骨拉麵

採用將豚骨長時間燉煮後萃取出的高湯作為湯頭使用。最大的特徵是有著獨特的香氣及濃厚的口感。最具代表性的是九州博多拉麵，最常可見的是白濁的湯頭。

Last updated 3 mins ago
鹽味拉麵

將以鹽進行調味所製的醬汁作為湯頭使用。與其他的拉麵相比，較能直接突顯出高湯本身的美味。由於味道清爽且容易入口，因此深受大眾喜愛。

Last updated 3 mins ago
海鮮系拉麵

將採用乾燥過後的柴魚、昆布及小魚乾等各式各樣海鮮所萃取出的醬汁作為高湯使用。其特徵為有著深層的美味及風味。

Last updated 3 mins ago
沾麵

將麵條過冷水後放入碗中拿給客人，客人再把麵條放入別的器碗中沾汁後享用。沾汁會比一般的拉麵湯頭的味道還要濃厚。店家會提供「加清湯（スープ割り）」的服務，當您將麵條享用完畢後，可以在沾汁裡加入豚骨清湯後品嚐美味的湯頭。

Last updated 3 mins ago
